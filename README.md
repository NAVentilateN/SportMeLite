# SportMe

A marketplace app that connects users and sports coaches.
[Live Demo](https://react-social-media-app-jet.vercel.app/)

## Features

* Add/Delete posts
* Comment on posts
* Follow/Unfollow other users
* Notifications
* Edit Profile
* Realtime messaging
* Light/Dark mode
* Mobile responsive
